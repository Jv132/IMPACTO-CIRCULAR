# üîë App Features

During sign in, you will be asked whether you are a Company or a Recycling Cooperative, and then proceed with the registration accordingly.

- Features for **Cooperatives**:
    - Waste management, Pickup and Logistics Data Records
- Features for **Companie**s:
    - Pickup Requests, Tracking and environmental reports.
- **Administrative** functions:
    - Validation and moderation of new users and dashboards with an overview of all ongoing collections.
- Future Features
    - We plan to offer courses about the waste management process so companies can implement them with their employees, since it's crucial for them to participate in the process.
